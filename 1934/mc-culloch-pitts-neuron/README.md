# McCulloch-Pitts Neuron (1934)

Proyecto de implementación de la neurona de McCulloch-Pitts, uno de los primeros modelos matemáticos de neurona artificial.

## Configuración del Entorno

### Activar el entorno virtual

**En Linux/Mac:**
```bash
source venv/bin/activate
```

**En Windows:**
```bash
venv\Scripts\activate
```

### Instalar dependencias

```bash
pip install -r requirements.txt
```

### Desactivar el entorno virtual

```bash
deactivate
```
